철봉 사진을 받아서 지도에 표시하는 Web app 을 만들고자 함.
- Desktop/Mobile web browser 지원
- HTML/JS/CSS 사용
- Leaflet 사용. 라이트 모드/다크 모드는 시스템 설정 따라가게
- Cloudfront 기능 이용하여 비용 최소화
- 최초로 접속하면 임의의 UUID 를 생성하여 local storage 에 저장하여 아이디로 사용
- 사진 파일을 선택하면 그걸 512x512 jpg 으로 resize 및 crop 하여 Cloudfront 서버에 저장. jpg 품질은 낮게 처리.
- 사진 파일에서 위도 경도 위치를 읽어서 Cloudfront 디비에 저장
- 서버 코딩은 TypeScript 또는 JavaScript 사용.
- 사진 파일을 CDN 에 올려야 비용이 최소화 되는 것으로 알고 있음
- 사용자 등급: 관리자, 파워유저, 일반 사용자
- 관리자와 파워유저는 사진 파일 업로드에 제한이 없음
- 일반 사용자는 하루 3개까지만
- 사용자 디비에는 관리자와 파워유저만 저장해두면 됨
- 일반 사용자가 올린 사진은 파워유저나 관리자가 승인할 때에만 공개됨
- 사진 디비를 관리해야 함. 위도 경도에 index 를 걸어서 사진 목록을 가져올 때 특정 위도 경도 내의 사진만 가져오기
- 이미 올린 곳과 10m 이내에 사진이 또 올라오는 경우 지도 표시할 때 같은 곳으로 처리.

- Cloudfront 관련 사항은 ../pi_finder 에서도 이용한 것을 참고하면 됨. 도메인은 pull.acidblob.com 으로 (아직 추가 안 됨)
- 구현 시작 전에 계획 공유 및 예상 비용 공유 후 구현 시작
- 추후 철봉 사진인지 아닌지 검토하는 파이프라인 붙일 예정
